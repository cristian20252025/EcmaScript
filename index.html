Tutorial ECMAScript

¿Qué es ECMAScript?

Cuando hablamos de ECMAScript, nos referimos al estándar que ya va por la versión ES6 y determina cómo emplear el lenguaje Javascript...

Versiones de ECMAScript

Como ocurre con todos los avances tecnológicos, Javascript ha pasado por diferentes versiones...

VersiónAñoNovedades
ES11997Primera versión
ES21998Cambios editoriales
ES31999
  • Se agregaron expresiones regulares.
  • Se agregó try/catch.
  • Se agregó switch.
  • Se agregó do-while.
ES4Nunca publicado
ES51999
  • Se agregó "modo estricto".
  • Se agregó compatibilidad con JSON.
  • Se agregó String.trim().
  • Se agregó Array.isArray().
  • Se agregaron métodos de iteración de matriz.
  • Permite comas finales para literales de objeto.
Ejemplos
ES62015
  • Se agregaron let y const
  • Se agregaron valores de parámetros predeterminados
  • Se agregó Array.find()
  • Se agregó Array.findIndex()
Ejemplos

Ejemplos ES5

La directiva de "uso estricto"

"use strict" define que el código JavaScript debe ejecutarse en "modo estricto".

Acceso a la propiedad en cadenas

var str = "HELLO WORLD";
str.charAt(0); // returns H

Matriz para cada uno()

var txt = "";
var numbers = [45, 4, 9, 16, 25];
numbers.forEach(myFunction);

function myFunction(value) {
  txt = txt + value + "
";
}

Ejemplos ES6

let y const

La palabra clave let permite declarar variables con alcance de bloque.

let txt = "";
let numbers = [45, 4, 9, 16, 25];
numbers.forEach(function(value) {
  txt += value + "
";
});

Constantes

const permite declarar una variable constante:

var x = 10;
// Aquí x es 10
{
  const x = 2;
  // Aquí x es 2
}
// Aquí x es 10

Transpiladores

Javascript está en constante cambio. Para mantener la compatibilidad entre versiones, se usan transpiladores como Babel o TypeScript...
